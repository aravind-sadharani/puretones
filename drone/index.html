PureTones Drone

PureTones Drone is a digital Tanpura with controls for fine tuning the strings and adjusting their harmonic envelope. PureTones Drone has been developed using Faust, a functional programming language for sound synthesis and audio processing. This enables skilled Indian Classical musicians to fine tune the harmonic envelope to suit whichever Raga they wish to perform or practice.

Contrary to widely held perceptions, the Tanpura is not only responsible for providing a reference of the fundamental note Sa, but it actually provides a tuning system resulting from string overtones which supports the entire Raga being performed. To understand this better, read this article which illustrates the role of the Tanpura in Indian Classical music.

Demo of PureTones Drone

Please check out the PureTones Drone Webapp for a demo of PureTones Drone. Here is a quick guide to use the webapp.

  1. Select your pitch using the + and − buttons for setting Pitch.
  2. Move the Play slider to the top to start playing a String.
  3. Use the String tabs to access each of the six strings.
  4. Use the Fine Tune and Ultrafine Tune sliders to finetune the pitch of each string.
  5. Use the Variance and Octave sliders to fine tune the harmonic content produced by each string.

Note that this is a demo application created using faust2webaudiowasm (see Faust Tools for details). It has been provided to help you understand the capabilities of PureTones Drone and is not intended for optimum performance and usability.
